## Props

| 名称         | 说明           | 类型                  | 默认值 |
| ------------ | -------------- | --------------------- | ------ |
| container    | 容器           | string \| HTMLElement |        |
| delay        | 延迟，单位毫秒 | number                | 100    |
| disabled     | 禁用           | boolean               | false  |
| overlayClass | 弹出层类名     | string                |        |
| overlayStyle | 弹出层样式     | object                |        |
| open         | 是否打开       | boolean               |        |
| title        | 标题           | string \| v-slot      |        |
